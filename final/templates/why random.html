About the Project

The project utilizes a dataset containing information such as age, gender, various blood tests, and other clinical parameters. These features are used to train the machine learning models and make predictions regarding liver disease.

Random Forest Model

Random Forest is a machine learning algorithm used for classification tasks. It is an ensemble learning method that combines multiple decision trees to make predictions. In our liver disease prediction project, we chose the Random Forest model due to its ability to handle high-dimensional datasets and handle non-linear relationships between features and the target variable.
